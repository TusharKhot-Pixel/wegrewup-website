# wegrewup website

Static landing page and privacy policy for [wegrewup](https://github.com/TusharKhot-Pixel/wegrewup-website) Android apps.

## Deploy on Vercel

1. Import this repository in [Vercel](https://vercel.com/new).
2. Framework preset: **Other** (static).
3. Deploy — no build command needed.
4. Use the deployed URL as your Play Console **Privacy policy** link.

Privacy policy anchor: `https://YOUR-VERCEL-URL/#privacy`
